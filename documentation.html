Face Recognition Attendance System Documentation

1. Registration Process

  1. Enter a numeric ID for the student
  2. Enter the student's name (letters and spaces only)
  3. Click "Take Images" to capture 30 face samples
  4. Click "Save Profile" to train the recognition model

2. Taking Attendance

  1. Click "Take Attendance" to start face recognition
  2. The system will automatically mark attendance for recognized faces
  3. Results will be displayed in the main window
